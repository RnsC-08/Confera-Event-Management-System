# Confera - Event Management System

<p align="center">
  <img src="assets/confera-icon.png" alt="Confera Logo" width="280"/>
</p>

Confera is a web-based Event Management System designed to help event centers manage their daily operations in a simple and organized way. The system focuses on event bookings, hall availability, client information, event services, equipment management, invoices, payments, and user roles.

The purpose of Confera is to reduce manual work, avoid double bookings, improve event organization, and provide a clear overview of scheduled events and available halls.

## What Confera Does

- Manages event bookings and reservations
- Checks hall availability
- Stores client and event organizer information
- Manages event services such as catering, decoration, and audio-visual equipment
- Tracks equipment and event resources
- Generates invoices and records payment status
- Supports user roles such as Administrator, Event Coordinator, and Operational Staff
