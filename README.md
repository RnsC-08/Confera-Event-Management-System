# Confera Event Management System


Confera is a web-based Event Management System for event centers and venues.
It helps teams manage event bookings, halls, clients, services, equipment, invoices, payments, staff assignments, reports, users, and role-based access from one central dashboard.

## Main Technologies

* Next.js
* React
* TypeScript
* MySQL
* Tailwind CSS
* pnpm

## Main Features

* Login and role-based access
* Dashboard overview
* Event booking management
* Event hall management
* Client management
* Services and service packages
* Equipment management
* Invoices and payments
* Staff assignments
* Reports
* Users management

## How to Run

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Open:

```text
http://localhost:3000/login
```

## Useful Commands

```bash
pnpm exec tsc --noEmit
pnpm build
pnpm dev
```
